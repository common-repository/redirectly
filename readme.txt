== Description ==
If only one post is found by the search results, redirectly will redirect the user to that post

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory, alternatively, just install this plugin from the WordPress plugin directory
2. Activate the plugin through the plugin menu in WordPress
